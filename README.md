# ERP 인사 길드

업로드한 **2026 에듀윌 ERP 정보관리사 인사 2급 파이널 핵심노트**를 기반으로 만든 모바일 중심 퀴즈 학습 게임입니다.

## 저장소 이름

```text
erp-hr-guild
```

## 주요 기능

- PDF 2~42페이지를 12개 챕터로 요약
- 핵심 카드와 플래시카드 용어 학습
- 객관식·OX·주관식·혼합 퀴즈
- 쉬움·보통·어려움·전체 난이도 선택
- 평균임금·소득금액·과세표준·일용근로 원천징수·공제액 계산 문제
- 틀린 문제를 약 3문제 뒤 주관식으로 자동 재출제
- 오답 목록과 챕터별 정답률, 최근 기록
- 레벨·경험치·골드 성장 요소
- 브라우저 `localStorage` 저장, JSON 백업·복원
- 모바일 하단 고정 메뉴와 큰 터치 버튼

## 실행

빌드 과정이나 프레임워크가 없는 정적 사이트입니다.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`을 엽니다.

## 검수

```bash
node scripts/validate.mjs
```

검수 항목:

- 문제은행 120문항
- 챕터 12개
- 문항 ID 중복 없음
- 객관식 정답이 보기 안에 존재
- 난이도 1~3 범위
- 필수 HTML 요소 존재
- `app.js`, `data.js` 문법 검사

## GitHub Pages 배포

1. GitHub에서 `erp-hr-guild` 저장소 생성
2. 이 폴더의 파일을 저장소 루트에 업로드
3. **Settings → Pages**
4. **Deploy from a branch** 선택
5. `main` 브랜치와 `/ (root)` 선택

## 자료 기준

- 문제 수: 120문항
- 출처 범위: PDF p.2-p.42
- 법령·세율·공제금액은 업로드한 2026 교재의 시험 기준입니다.
- 실제 신고·노무 업무에는 최신 공식 법령을 별도로 확인해야 합니다.

원문 PDF 자체는 저작권 문제를 고려해 저장소에 포함하지 않았습니다. 문제은행 유지보수용 엑셀 파일만 `source/`에 포함했습니다.
