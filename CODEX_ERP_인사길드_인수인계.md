# ERP 인사 길드 프로젝트 인수인계

## 저장소

- 권장 이름: `erp-hr-guild`
- 정적 GitHub Pages 프로젝트
- 프레임워크·빌드 과정 없음
- 저장 키: `erpHrGuildV1`

## 목적

2026 ERP 정보관리사 인사 2급 파이널 핵심노트를 모바일에서 반복 학습하기 위한 RPG형 퀴즈 게임이다.

## 파일 구조

- `index.html`: 화면과 모바일 UI
- `data.js`: 검수된 120문항, 12개 챕터, 학습 카드
- `app.js`: 퀴즈·오답·계산·기록·저장 로직
- `source/*.xlsx`: 유지보수용 문제은행
- `scripts/validate.mjs`: 정적 검수
- `README.md`: 실행 및 Pages 배포 방법

## 구현된 기능

- 챕터 요약, 핵심 카드, 플래시카드
- 객관식, OX, 주관식, 혼합 문제
- 난이도 하·중·상·전체 선택
- 10·20·30문제 선택
- 7종 랜덤 계산 문제
- 틀린 문제 약 3문제 뒤 주관식 재출제
- 계산 문제도 같은 수치로 재출제
- 오답 목록, 최근 기록, 챕터 숙련도
- 레벨, XP, 골드
- JSON 백업·복원
- 모바일 하단 고정 메뉴

## 계산 문제 기준

- 평균임금
- 소득금액
- 종합소득 과세표준
- 일용근로자 원천징수세액
- 기본공제액
- 자녀세액공제액
- 외국납부세액공제 한도액

모든 공식과 금액은 업로드된 PDF 기준이다.

## 검수 명령

```bash
node --check app.js
node --check data.js
node scripts/validate.mjs
```

브라우저 스모크 테스트 완료 항목:

- 모바일 390×844 화면 로드
- 12개 챕터 렌더링
- 객관식 제출과 다음 문제
- 주관식 오답 재출제 예약
- 계산기 계산 결과 복사
- 난이도 상 계산 문제 출제
- 계산 문제 3문제 뒤 재출제
- 기록 반영
- 콘솔 오류 없음

## 이후 개선 후보

1. 실제 기출문제와 실무 입력형 문제 추가
2. 챕터별 보스와 업적 추가
3. 문제은행을 별도 JSON 파일로 분리
4. PWA 설치 지원
5. 실제 휴대폰 QA 후 여백과 키보드 대응 보정

원문 PDF를 임의로 재배포하지 말고, 문제 내용 수정 시 PDF 페이지와 문제은행 엑셀을 함께 대조할 것.
